购车建议

通用选择规则（按真实成本）

1. 年里程 ≥ 2 万公里 + 家充为主 + 用车 ≥ 5 年 → 电车更划算
2. 年里程 ＜ 1.5 万公里 或 车价 ≥ 25 万 → 油车更划算
3. 高速为主 + 外面充电 → 优先油车
4. 油价每上涨0.5元/L，油车年均成本增加约400-500元（1万公里）
5. 无论什么条件，最终以本工具实际计算成本为准，谁低选谁

参数计算说明

一、折旧费用（核心成本）
计算逻辑
  • 年折旧费用 = (购车总价 - 车辆残值) ÷ 用车年限
  • 车辆残值 = 购车总价 × 对应年限残值率
    • 残值率标准（市场平均水平）
  • 燃油车：3年70%、5年60%、8年45%、10年30%、15年15%
  • 纯电车：3年55%、5年45%、8年30%、10年20%、15年8%
    • 二、能源成本（油费/电费）
    燃油车油费
  • 年油费 = (年里程 ÷ 100) × 车型油耗 × 自定义汽油单价
  • 油耗标准（市区/高速）：
    - 紧凑型：8.5L/100km / 7.0L/100km
    - 中型车：9.5L/100km / 7.8L/100km
    - SUV：11L/100km / 9.0L/100km
    • 纯电车电费
  • 年电费 = (年里程 ÷ 100) × 车型电耗 × 自定义电价
  • 电耗标准（市区/高速）：
    - 紧凑型：11度/100km / 14度/100km
    - 中型车：14度/100km / 17度/100km
    - SUV：16度/100km / 20度/100km
    • 三、保养费用
  • 年保养费 = 基础保养费 + (年里程 ÷ 10000) × 每万公里额外费用
  • 燃油车：基础1500元/年 + 300元/1万公里
  • 纯电车：基础500元/年 + 100元/1万公里（无发动机/变速箱，成本更低）
    • 四、保险费用
  • 年保险费 = 购车总价 × 保险费率
  • 燃油车：1.8%/年（行业平均水平）
  • 纯电车：2.2%/年（电车保费略高）
    • 五、税费成本
    购置税（仅燃油车）
  • 总购置税 = 购车总价 ÷ 1.13 × 10%，分摊到用车年限
  • 纯电车目前免征购置税
    • 车船税
  • 燃油车：360元/年（1.6-2.0L排量标准）
  • 纯电车：免征车船税
    • 六、电池更换成本（仅纯电车）
  • 仅用车年限≥8年时计算，总更换成本=购车总价×30%
  • 年电池成本 = 总更换成本 ÷ 用车年限
  • 电池成本标准：约占车价30%（市场平均更换价）
    • 七、总成本构成
  • 燃油车总成本 = 折旧 + 油费 + 保养 + 保险 + 购置税分摊 + 车船税
  • 纯电车总成本 = 折旧 + 电费 + 保养 + 保险 + 电池更换成本（8年以上） + 车船税（0）
